This Power BI project analyzes customer loss data to identify patterns and reasons behind churn. The dashboard provides clear visuals to help businesses understand retention through demographic and service-related factors.

Data Dimensions (Inside the Report)
The analysis is built around the following actual data fields:

Demographics: Gender.
Customer Profiling: Identifying which age groups and service plans have the highest churn.
Subscription Type: Basic- Standard -Premium
Contractual Length:Monthly- Quartarly -Annual
Avrege of delay days (how many months or days they stay without payment before leaving the service)

Key Insights Delivered
Churn Rate: A clear metric showing the percentage of customers who left.
Financial Impact: Monitoring revenue loss from churned customers compared to remaining revenue.
Power Query: Cleaning and transforming the raw customer data.

DAX: Custom measures for calculating Churn Rate and total counts.
